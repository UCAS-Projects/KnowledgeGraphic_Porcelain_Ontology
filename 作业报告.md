# 语义网与知识图谱作业：瓷器知识库本体建模

## 1. 作业目标
基于OWL、RDF和XML的技术标准，熟练运用语义网技术建立一个关于“瓷器”的领域本体（Ontology）知识库。

## 2. 数据素材
作业采用了一份手工构建的典型中国传统瓷器小型数据集作为素材（详见 `porcelain_dataset.csv`），包含以下五件极具代表性的瓷器：
1. 北宋汝窑天青釉洗
2. 元青花鬼谷子下山图罐
3. 明成化斗彩鸡缸杯
4. 清乾隆粉彩百鹿尊
5. 宋代建窑黑釉兔毫盏

## 3. 本体设计与知识建模

由于需要满足标准的由顶向下知识建模，本次设计抽取了以下概念（Classes）与属性（Properties）：

### 3.1 核心类 (Classes)
- **Porcelain (瓷器)**：顶级实体概念。
- **Dynasty (朝代)**：宋代、元代、明代、清代等。
- **Kiln (窑口)**：汝窑、景德镇窑、建窑等。
- **GlazeColor (釉色)**：青釉、青花、斗彩、粉彩、黑釉等。
- **Shape (器型)**：洗、罐、杯、尊、盏等。

### 3.2 对象属性 (Object Properties)
表示类与类实例之间的逻辑关系：
- `hasDynasty`：瓷器 -> 朝代（Domain: Porcelain, Range: Dynasty）
- `hasKiln`：瓷器 -> 窑口（Domain: Porcelain, Range: Kiln）
- `hasGlaze`：瓷器 -> 釉色（Domain: Porcelain, Range: GlazeColor）
- `hasShape`：瓷器 -> 器型（Domain: Porcelain, Range: Shape）

### 3.3 数据属性 (Data Properties)
表示实例的具体属性值：
- `hasName`：瓷器的具体名称 (类型：xsd:string)
- `hasDescription`：瓷器的描述信息 (类型：xsd:string)

## 4. 技术标准与实现 (XML/RDF/OWL)

最终输出的本体文件为 `porcelain_ontology.owl`，该文件完全遵循W3C语义网技术标准，采用 **RDF/XML** 格式编写，能够被 **Protégé** 等主流本体编辑工具直接解析并加载。

- **XML标准**：利用 `<rdf:RDF>`, `<owl:Ontology>` 等XML标签结构化规范数据。
- **RDF标准**：通过 `rdf:about`, `rdf:resource` 及命名空间 (如 `xmlns:rdf`) 定义资源及其 URI，形成了典型的 “主-谓-宾” 三元组图结构。
- **OWL标准**：使用了 `<owl:Class>`, `<owl:ObjectProperty>`, `<owl:NamedIndividual>` 等标签定义丰富的逻辑和推理基础，比如定义互斥关系等（本作业暂保留基础分类）。

## 5. 如何使用
1. 下载并安装本体编辑工具 **Protégé**。
2. 打开 Protégé，点击 `File -> Open`。
3. 选择本目录下的 `porcelain_ontology.owl` 文件。
4. 在 Protégé 面板中，你可以分别在 `Entities` 下的 `Classes`, `Object properties`, `Data properties`, 和 `Individuals by class` 中查看完整的树状结构和实例图谱。
